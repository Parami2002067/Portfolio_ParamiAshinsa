Camera Rental Management System

Business Analysis Case Study

Project Overview

Camera Rental Management System is a Business Analysis project designed to digitize and improve the manual camera rental process by managing inventory, bookings, customers, and returns efficiently.

Problem Statement

Manual rental systems often cause booking conflicts, poor tracking of available equipment, and inefficient customer management, leading to operational delays and loss of revenue.

Stakeholders & Scope

  • Admin / System Owner
  • Customers (Renters)
  • Inventory Manager

Requirements

Functional Requirements

ID Requirement Description
FR-01 Manage Camera Inventory System allows admin to add, update, and remove camera equipment details.
FR-02 Customer Registration System allows users to register and maintain customer profiles.
FR-03 Camera Booking Customers can check availability and book cameras for selected dates.
FR-04 Return Management System records camera returns and updates inventory status.
FR-05 Penalty Calculation System calculates late return charges based on rental duration rules.

Non-Functional Requirements

  • System should respond within 2 seconds
  • Secure login for admin access
  • High availability during business hours
  • User-friendly interface

Use Case Scenarios

Scenario 1: Rent Camera

  • User selects available camera
  • System checks availability
  • Booking is confirmed

Scenario 2: Return Camera

  • User returns camera
  • System updates inventory
  • Late fees calculated if needed

Business Rules

Business Rules

Rule ID Business Rule Description
BR-01 No Double Booking A camera cannot be booked by multiple customers for the same time period.
BR-02 Registered Users Only Only registered customers are allowed to rent cameras.
BR-03 Return Deadline Enforcement All rentals must be returned within the agreed rental period.
BR-04 Late Fee Policy Late returns are charged based on a predefined penalty rate.
BR-05 Admin Approval High-value equipment rentals require admin approval before confirmation.
