Camera Rental Management System

Business Analysis Case Study

Executive Summery

Background

Many customers face difficulties when renting cameras and related equipment due to limited visibility of available equipment, manual booking processes and lack of booking history management.

Proposed Solution

A camera rental management system that enables users to browse available cameras, make bookings, track rental history, and manage reservations through a mobile application.

Expected Benefits

  • Faster booking process
  • Improved user experience
  • Better inventory management
  • Reduced manual workload

Business Problem

Current Challenges

  • Customers need to contact the renter manually
  • Availability information is difficult to update and track
  • Rental records are difficult to track
  • Booking conflicts may occur

Impact

  • Time consuming operations
  • Risks in managing bookings
  • Customer dissatisfaction

User Stories

  • As a customer, I want to create a user account as I can access to system and it’s services
  • As a customer, I want to log in to the system as I can access booking and account management features
  • As a customer, I want to search for cameras by category so that I can quickly find equipment that matches my needs.
  • As a customer, I want to book equipment through the system so that reserve equipment quickly and conveniently
  • As a customer, I want to cancel bookings so that I can modify or correct my reservation when necessary.
  • As a customer, I want to view my rental history so that I can track my previous rentals and bookings
  • As a customer, I want to edit profile so that my account details remain accurate and up to date
  • As a customer, I want to view camera details so that I can compare available equipment before making a booking
  • As a customer, I want to check equipment availability so that I can book only available items
  • As a customer, I want to log out of the system so that my account remains secure when using shared devices
  • As an administrator, I want to add new equipment so that customers can view newly available rental items
  • As an administrator, I want to update equipment details so that details remain accurate and up to date
  • As an administrator, I want to manage bookings so that rental operations remain organized
  • As an administrator, I want to view rent history so that I can monitor customer rentals and manage rental records effectively
  • As an administrator, I want to remove equipment from the inventory so that customers can view up to date and available equipment details only

Business Rules

ID Business Rule
BR-01 Users must register and authenticate before accessing protected system features
BR-02 Each user must register with a unique email address and username
BR-03 The password must contain at least 8 characters including letters, numbers and symbols
BR-04 Users may only book equipment that is available for the selected rental period
BR-05 Only administrators can add, update or delete equipment
BR-06 Only administrators may generate monthly rental reports
BR-07 The rental end date must be later than the rental start date
BR-08 A customer may not have overlapping bookings for the same rental period
BR-09 Cancelled bookings shall release the equipment for future reservations
BR-10 Users may edit only their own profile information
BR-11 Deleted user accounts shall no longer be able to access the system
