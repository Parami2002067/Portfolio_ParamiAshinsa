Portfolio Website

Frontend Development Case Study

Project Overview

This portfolio website was designed and developed to showcase my skills, projects, and professional profile as an ICT undergraduate focusing on Business Analysis, Data Analysis, and UI development.

Objective

To build a personal branding platform that presents my technical skills, academic projects, and career focus in a structured and visually engaging manner.

Key Features

Feature Description
Responsive Design Fully responsive layout for desktop and mobile devices.
Animated Hero Section Dynamic typing effect introducing roles and identity.
Project Showcase Structured case study style project presentation system.
Modern UI Design Glassmorphism style cards and dark theme aesthetics.
Navigation System Smooth scroll-based navigation between sections.

Technologies Used

HTML5 CSS3 JavaScript Font Awesome

Design Approach

The design follows a modern dark UI theme with soft neon accents, focusing on readability, spacing, and structured hierarchy. The goal was to create a clean recruiter-friendly interface rather than a decorative portfolio.

Outcome

The final result is a professional portfolio system that effectively communicates my skills, projects, and career direction in Business Analysis and Data Analysis.
